# aic_tikz_fig
Tikz figures for advanced integrated circuit course

# Things you need
- [Docker](http://www.docker.com)
- Make 
  - Linux: should be there already, if not find a gnumake package
  - Mac: Install build tools
  - Windows: Don't use. Install WSL2


# Getting started

```
clone
cd docker
make
cd ..
make run-sh
cd tex
make
```
